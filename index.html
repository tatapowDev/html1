Introdução

História

O primeiro computador surgiu em 1994, o Harvard Mark I, um computador eletro-mecânico, usado como uma "máquina cálculo". Anteriormente ja se ouvia sobre a criação de computadores, mas o Mark, foi de fato, oprimeiro a ser criado, justamente para cálculos.

Em 1946, surgi o Eniac, um computador automático, dos EUA, e foi exatamente nesta época, que sugiu o termo "BUG", que, devido os insetos, defecarem nas máquinas, aconteciam problemas técnicos. E aprtir de 1965, surgem os "circuitos integrados", os famosos "chps", diminuindo ainda mais o tamanho dos computadores, que até então, eram enormes.

Chegando então, apartir de 1971, novas tecnologias vão aparecendo até os dias atuais, como por exemplo, os "microprocessadores", compactando ainda mais os computadores, sendo denominados, como a quarta geração de computadores.

Vale ressaltar, que atualmente, estamos adentrando na quinta geração dos computadores, com o surgimento das IA´s, Metaverso, Cripto-moedas e também, os computadores quânticos.

Surgimento da Internet

A Internet é um mecanismo de comunicação, troca de informações. Mas até evoluir, até os dias atuais, partiremos do inicio de tudo, falando sobre um dos primeiros mecanismos de comunicação ,que foi o telégrafo e o código morse. Em 1958, surgiu a comunicação "cabeada", por meio de longos cabos, conectados de ponta-a-ponta, eram feitas as trocas de informações.

Em 1969, foi estabelecida a primeira conexão por "hipertexto" entre duas universidades, apenas com a palavra "LOGIN", e partir desta pequena conexão, surgiu o TCP/IP, fazendo com que Tim Berners-Lee, unisse o hipertexto com TCP/IP, criando assim o World Wide Web - WWW., e também o "Pai da internet."

Client Vs Server

O Client é o dispostivo que esta consumindo as informações da Internet, ou seja, desktop´s, notebook´s, smartphones ..., já o Server é quem sede as informações para os Client´s. Ou seja, quando hospedamos um site, estamos guardando no Server (servidor), para distribuir aos Client´s (Clientes).

Para melhor entender o fluxo: "O usuario através de um dispostivo, manda as informações para a Internet, pelo Navegador, gera-se um protocolo HTTP, que é enviado uma requisição para o Server. O Server, processa essa requisição, devolve ao navegador (que gera o "cache"),onde armazena essas dados no dispositvo do Client, que repassa os dados para o usuário.

O Server, retorna ao navegador, arquivos: HTML, CSS, JAVASCRIPT, Imagens..., chamados de arquivos estáticos ( ou seja, arquivos que saem do Server para o navegador)

Um ponto muito importante, é a questão da "RESPONSIVIDADE" e "COMPATIBILIDADE", ou seja, quem trabalha na parte Client de uma aplicação WEB, ter ciencia para estruturar essas informações, para serem visualizadas de diferentes dispostivos, com Boas Práticas e Testes, pois cada navegador, se comporta de uma determinada maneira.

Navegadores (ou "BROWSERS"), são programas instalados no dispositvo, para INTERPRETAR os arquivos que vem do Servidor web. Bem como: Google Chrome, Safari, Microsoft Edge, Mozila Firefox...

Linguagem de Programação

Definição: "É uma linguagem escrita e formal que especifica um conjunto de instruções e regras usadas para gerar programas (softwares).

Algumas linguagens de programação: JavaScripti, Python, Java, PHP, C#, entre outras.

Linguagem Server-Side são linguagens que rodam do lado do Servidor (JavaScripti, Python, Java, PHP, C#, entre outras), ou seja, é nele que é feita toda a LÓGICA e PROCESSAMENTO.

Linguagem Client-Side são linguagem que rodam do lado do cliente (dispositivo), como o JavaScript

HTML não é linguagem de programação! HTML é um arquivo de MARCAÇÃO, ou seja, é o que chamamos de "esqueleto" do nosso projeto em questão. São textos delimitados por nomes que o navegador consegue interpretar, e cada um desses nomes tem FUNÇÕES e COMPORTAMENTOS específicos, as chamadas T A G´s.
